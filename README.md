# TNT

**TNT (Tnt is Not TradingView)**

An open-source trading and market analysis platform written in Python.

## Planned Features

* Multiple data providers
* Multiple candle types
* Custom indicators
* Custom strategies
* Replay engine
* Backtesting
* Paper trading

## Built With

* Python
* NiceGUI
* TradingView Lightweight Charts

## Status

Early development.
